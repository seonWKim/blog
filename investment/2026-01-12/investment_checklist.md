▶ LOGIC STRONGER 일 때

**기본 행동:**
- 자산 유형에 따라 차등 적용

**A. 인프라 자산 (AEP, DUK, GEV):**
- 목표 비중: +2~3%p (최대 +5%p까지)
- DCA 가중: +15~25% (1개월 한정)
- 조건: CAPEX 회수 구조 명확 + 규제 리스크 낮음 확인

**B. 사이클 자산 (TER, FORM):**
- 목표 비중: +1~2%p (최대 +3%p까지)
- DCA 가중: +10~20% (1개월 한정)
- 조건:
  * 실제 주문 증가 확인 (백로그만으로는 부족)
  * 고객 다각화 또는 장기 계약 존재
  * 마진 유지/개선 확인
- 주의: 사이클 자산은 빠른 반전 가능성 → 보수적 접근

**C. 헤지 자산 (GLD):**
- 목표 비중: 현재 비중 유지 OR 최대 +1~2%p
- DCA: 유지 (가중 금지)
- 조건:
  * 고점 근처에서는 비중 확대 금지
  * 포트폴리오 변동성 헤지 목적 유지
  * 리스크 온(risk-on) 전환 시 즉시 축소 준비
- 원칙: "헤지는 가격이 올라서가 아니라 리스크가 커져서 늘린다"

**D. 옵션 자산 (SMR):**
- 목표 비중: 3~5% 범위 내 유지
- DCA: 조건부 매수만 (월 1회, 조건 충족 시)
- 조건:
  * 구속력 있는 PPA 체결 확인
  * 자금 조달 리스크 감소 확인
  * LOGIC STRONGER만으로는 비중 증액 불가

**추가 지시:**
- 어디서 논리가 더 강해졌는지 1~2줄로 명확히 설명
- 증거 체크리스트(A/B/C) 중 몇 개 충족했는지 명시
- "STRONGER" 판정에도 불구하고 비중을 크게 올리지 않는 이유 설명

▶ LOGIC INTACT 일 때

**행동:**
- 아무것도 하지 말 것
- 기존 DCA 그대로 유지
- 목표 비중 변경 없음

**추가 지시:**

- 다음 STRONGER/WEAKENING으로 전환될 가능성 있는 트리거 언급

▶ LOGIC WEAKENING 일 때

**기본 행동:**
- 극단적 조치(0% 또는 전량 매도) 금지
- 단계적 축소 원칙

**A. 신규 매수 중단 (PAUSE):**
- 즉시 신규 매수 중단
- 기존 보유분은 유지

**B. 목표 비중 단계적 축소:**
- 1차 조정: -3~5%p (예: 15% → 10~12%)
- 2차 조정: 다음 점검에서 WEAKENING 지속 시 추가 -3~5%p
- 최종: 최소 유지 비중 5% (완충재 역할)

**C. 예외: 완충재 자산 (DLR 같은 현금흐름 자산):**
- WEAKENING이라도 즉시 0%로 가지 말 것
- 1차: 15% → 8~10% (완충 기능 유지)
- 2차: 다음 점검에서 BROKEN 확인 시 5% 또는 0%

**추가 지시:**
- 약화 신호가 '일시적 사이클'인지 '구조적 변화 가능성'인지 구분
- 포트폴리오 전체에서 이 축소가 집중도를 악화시키지 않는지 확인
- 다음 점검 때 확인해야 할 '결정적 트리거' 제시

---

▶ LOGIC BROKEN 일 때

**행동:**
- 즉시 전량 매도 ❌
- 아래 중 하나 이상을 제안:
    1) 2~3회 분할 매도 시나리오 (각 -5~10%p씩)
    2) 자연 비중 축소 (다른 종목만 매수, 해당 종목 매수 중단)
    3) 대체 병목 자산으로 전환 (유사 섹터 내 다른 종목)

**우선순위:**
- 1순위: 자연 비중 축소 (세금 효율)
- 2순위: 대체 자산 전환 (섹터 익스포저 유지)
- 3순위: 분할 매도 (손실 확정)

**추가 지시:**
- "무엇이 어떻게 바뀌어서 논리가 깨졌는지"를 구조적으로 설명
- 감정적 표현 금지
- 대체 투자 아이디어 제시 (섹션 8과 연계)
