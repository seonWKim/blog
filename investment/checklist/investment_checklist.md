## 2) 논리 상태 분류 + 밸류에이션 체크 ★ UPDATED

각 종목을 아래 4단계 중 하나로 분류하되, **반드시 밸류에이션도 동시 평가해라**

### LOGIC STRONGER 판정 기준 (기존 유지)

A. 증거 품질 (최소 2개 이상 충족)
   □ 회사 공시/IR/실적콜에서 직접 언급
   □ 구속력 있는 계약 체결 (백로그가 아니라 확정 주문)
   □ 신뢰도 높은 1차 출처 (Reuters, Bloomberg, WSJ 등)
   □ 재무제표상 매출/마진 개선 확인

B. 수익 전환 구조 (병목이 기업 이익으로 전환되는 메커니즘)
   □ 가격 결정력 확인 (병목 → 가격 인상 가능)
   □ 비용 회수 구조 명확 (규제/계약 보장)
   □ CAPEX 부담 < 매출 증가 (특히 유틸리티)

C. 지속성 (일회성이 아닌 구조적 변화)
   □ 최소 2~3분기 이상 지속 가능한 동력
   □ 경쟁사 진입장벽 존재
   □ 고객 다각화 (단일 고객 의존도 < 30%)

**출처 신뢰도 등급 (STRONGER 판정 시 필수 확인)**
- Tier 1 (강력): 회사 공시, Reuters, Bloomberg, WSJ, FT
- Tier 2 (중간): Industry 리포트 (TrendForce, Gartner), SeekingAlpha Pro
- Tier 3 (약함): Medium, Stocktwits, 일반 블로그
→ STRONGER 판정은 Tier 1 출처 2개 이상 OR Tier 1+Tier 2 조합 필요

### ★ NEW: 밸류에이션 체크포인트 (LOGIC STRONGER 종목 필수)

**모든 LOGIC STRONGER 종목에 대해 아래 3가지 질문에 답해라:**

#### 1. What's Priced In? (무엇이 이미 주가에 반영되었는가?)

**확인 항목:**
- **Consensus Estimates:**
  - 향후 2년 EPS 컨센서스 (검색: Yahoo Finance, Seeking Alpha)
  - 향후 2년 매출 성장률 컨센서스

- **Multiple Expansion:**
  - 현재 Forward P/E: [검색]
  - 섹터 평균 Forward P/E: [검색]
  - 5년 평균 Forward P/E: [검색]
  - 판정: 멀티플이 이미 확장되었는가? (현재 P/E > 5년 평균 × 1.2)

- **Street Sentiment:**
  - Analyst Ratings: Buy/Hold/Sell 비율 (검색)
  - Recent Upgrades: 최근 30일 내 목표가 상향 건수
  - 판정: 이미 과도한 낙관론인가? (Buy >70% + 연속 상향)

**예시:**
```
GEV 밸류에이션 체크:
- Forward P/E: 28x (섹터 평균 15x, 5년 평균 18x)
- 판정: ⚠️ EXPENSIVE - 멀티플 이미 56% 프리미엄
- 컨센서스 EPS 성장률: 15% (향후 2년)
- 암묵적 성장률: 28 P/E × 15% growth = PEG 1.87 (>1.5는 비쌈)
- 결론: 논리는 STRONGER이나 가격은 이미 상당 부분 반영됨
```

#### 2. Margin of Safety (안전마진은 얼마인가?)

**DCF Fair Value 간이 계산:**

```
Fair Value 추정 (간단 버전):
1. 내년 예상 EPS × (1 + 성장률)^4 = 5년 후 EPS
2. 5년 후 목표 P/E (보수적: 섹터 평균 × 0.9)
3. 5년 후 목표가 = 5년 후 EPS × 목표 P/E
4. 현재가로 할인 (10% 할인율): 목표가 / (1.10)^5
5. Margin of Safety = (Fair Value - 현재가) / 현재가

예시:
- GEV 현재가: $350
- 내년 EPS: $8 → 5년 후: $8 × (1.12)^4 = $12.6
- 목표 P/E: 15x × 0.9 = 13.5x
- 5년 후 목표가: $12.6 × 13.5 = $170
- 현재 Fair Value: $170 / (1.10)^5 = $106
- Margin of Safety: ($106 - $350) / $350 = -70% ⚠️ OVERVALUED
```

**단순 비교법:**
- **PEG Ratio:** Forward P/E ÷ 성장률
  - <1.0: Undervalued
  - 1.0-1.5: Fair
  - >1.5: Overvalued

- **EV/EBITDA vs Peers:**
  - 현재 EV/EBITDA: [검색]
  - 동종업계 평균: [검색]
  - 판정: 프리미엄/디스카운트 %

- **Price/Sales vs Historical:**
  - 현재 P/S: [검색]
  - 5년 평균 P/S: [검색]
  - 판정: 역사적 평균 대비 위치

#### 3. Expected Return (기대수익률은 충분한가?)

**12-18개월 목표가 산출:**

```
시나리오 분석:

Bear Case (확률 20%):
- 논리 약화, 실적 부진
- 목표 EPS: $7, 목표 P/E: 12x
- 목표가: $84 (-76% from $350)

Base Case (확률 50%):
- 논리 유지, 컨센서스 달성
- 목표 EPS: $9, 목표 P/E: 15x
- 목표가: $135 (-61% from $350)

Bull Case (확률 30%):
- 논리 강화 지속, 서프라이즈
- 목표 EPS: $11, 목표 P/E: 18x
- 목표가: $198 (-43% from $350)

확률가중 기대가: $84×0.2 + $135×0.5 + $198×0.3 = $143
Expected Return: ($143 - $350) / $350 = -59%
Risk/Reward: -59% / +43% = -1.4 (매우 불리)
```

**판정 기준:**
- Expected Return >30%: 강력 매수 (비중 증액 적극)
- Expected Return 15-30%: 매수 (비중 증액 보수적)
- Expected Return 0-15%: 홀드 (비중 동결, 신규 매수 최소)
- Expected Return <0%: 회피 또는 축소

**최종 규칙:**
```
IF (LOGIC STRONGER) AND (Expected Return <15%):
   → 비중 증액 금지, 최대 유지만 가능

IF (LOGIC STRONGER) AND (Expected Return >30%):
   → 비중 증액 적극 (자산 유형별 규칙 적용)
```

## 5) 포지션 사이징 수학 ★ NEW

### Kelly Criterion 적용 (Modified Half-Kelly)

**공식:**
```
Kelly % = (Win Rate × Avg Win - Loss Rate × Avg Loss) / Avg Win
Position Size = Kelly % ÷ 2 (안전마진)
```

**각 종목별 계산:**

**예시: GEV**
```
시나리오 확률:
- Bull: 30% 확률, +60% 수익
- Base: 50% 확률, +20% 수익
- Bear: 20% 확률, -30% 손실

기대값 = 0.3×0.60 + 0.5×0.20 + 0.2×(-0.30) = 0.22 (22%)

Win Rate = 80% (Bull + Base)
Avg Win = (0.3×60 + 0.5×20) / 0.8 = 35%
Loss Rate = 20%
Avg Loss = 30%

Kelly = (0.80 × 0.35 - 0.20 × 0.30) / 0.35 = 63%
Half-Kelly = 31.5%

조정 (제약 고려):
- 유동성: OK (대형주)
- 상관관계: 다른 전력주와 0.7 상관 → -5%p 조정
- 섹터 한도: 전력 45% 제한 고려

최종 포지션: 26% → 현실적 목표 12-15% (포트폴리오 균형)
```

**실전 적용:**
1. 각 종목 시나리오 분석 → 기대수익 계산
2. Kelly Criterion → 이론적 최적 비중
3. 제약 조건 반영:
   - 섹터 한도
   - 상관관계
   - 유동성
   - 개인 리스크 성향
4. 최종 목표 비중 결정

**간편 버전 (빠른 판단용):**
```
High Conviction (Expected Return >40%, 확률 >70%):
→ 목표 비중: 15-20%

Medium Conviction (Expected Return 20-40%, 확률 60-70%):
→ 목표 비중: 10-15%

Low Conviction (Expected Return 10-20%, 확률 50-60%):
→ 목표 비중: 5-10%

Very Low Conviction (Expected Return <10% or 확률 <50%):
→ 목표 비중: 0-5% (또는 홀드만)
```

```
- 보유 11개월 + 이익 +50% + 매도 고려 중
  → 1개월 더 기다려서 장기세율 적용

예시:
GEV 보유 350일, +60% 이익
- 지금 매도: $10k 이익 × 30% = $3k 세금
- 15일 후 매도: $10k 이익 × 15% = $1.5k 세금
- 절감액: $1.5k
```
