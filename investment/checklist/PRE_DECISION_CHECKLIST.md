# Pre-Decision Checklist
**목적:** 중요한 투자 결정 전 체계적 검증 및 인지 편향 방지

**사용 시점:** 다음 조건 중 하나 이상 충족 시 필수 실행
- 단일 종목 비중 ±5%p 이상 조정
- $10,000 이상 일시불 투입/매도
- 신규 종목 진입 (기존 포트폴리오 외)
- 논리 상태가 2단계 이상 변화 (예: INTACT → WEAKENING)
- 감정적으로 동요되었다고 느낄 때

---

**결정 ID:** [YYYY-MM-DD-TICKER-ACTION]

**결정 내용:**
```

**현재 심리 상태:**
- [ ] 평온함 (정상)
- [ ] 흥분/기대감 (FOMO 가능성 - 주의)

**시장 환경:**

- 해당 종목 최근 7일: [+/- Z%]

## PART 2: 인지 편향 체크 (Cognitive Bias Checks)

### 2.1 앵커링 편향 (Anchoring Bias)
**질문:** 최근 가격/뉴스에 과도하게 집착하고 있는가?

- [ ] ✅ 아니오 - 장기 데이터 기반 판단
- [ ] ⚠️ 예 - 다음 질문에 답하시오:

**만약 예라면:**
1. **현재 앵커:** 무엇에 집착하는가? [기록]
2. **장기 관점:** 6개월 전 데이터는? 12개월 전은? [조사]
3. **재평가:** 장기 관점에서도 결정이 타당한가? [Yes/No]

### 2.2 확증 편향 (Confirmation Bias)
**질문:** 내 의견을 지지하는 정보만 찾고 있는가?

- [ ] ✅ 아니오 - 반대 의견도 적극 검토함
- [ ] ⚠️ 예 - 다음을 수행하시오:

**만약 예라면:**
1. **Steel Man 반론:** 내 결정에 반대하는 가장 강력한 논거는?
   ```
   [반대 논거 3개 이상 기록]
   -
   -
   -
   ```

2. **반론에 대한 답변:** 위 반론을 어떻게 반박하는가?
   ```
   [각 반론에 대한 구체적 반박]
   -
   -
   -
   ```

3. **판정:** 반박이 충분히 강력한가? [Yes/No]

### 2.3 FOMO (Fear of Missing Out)
**질문:** 다른 사람들이 돈을 벌고 있어서 조급한가?

- [ ] ✅ 아니오 - 내 논리와 타이밍 기반
- [ ] ⚠️ 예 - **24시간 대기 필수**

**만약 예라면:**
1. **대기 기간:** 24시간 후 재검토 날짜: [YYYY-MM-DD HH:MM]
2. **냉각 질문:**
   - 지금 사지 않으면 영원히 기회가 사라지는가? [Yes/No]
   - 1주일 후에도 이 기회는 유효한가? [Yes/No]
   - 급등 후에 진입하는 것은 아닌가? [Yes/No]

### 2.4 손실 회피 편향 (Loss Aversion)
**질문:** 손실을 확정하기 싫어서 판단이 흐려졌는가?

- [ ] ✅ 아니오 - 손익과 무관한 판단
- [ ] ⚠️ 예 - 다음을 수행하시오:

**만약 예라면:**
1. **Sunk Cost 확인:** 현재 손익: [+/- X%], 금액: $[금액]
2. **제로베이스 질문:** "내가 이 종목을 보유하지 않았다면, 지금 이 가격에 살 것인가?"
   - [ ] Yes → 보유/증액 타당
   - [ ] No → 매도 고려
3. **최악 시나리오:** 지금 손절하지 않고 더 하락하면? [-X% 추가 하락 시 총 손실: $Y]

### 2.5 과신 편향 (Overconfidence Bias)
**질문:** 내가 시장보다 더 잘 안다고 확신하는가?

- [ ] ✅ 아니오 - 불확실성 인정
- [ ] ⚠️ 예 - 다음을 수행하시오:

**만약 예라면:**
1. **과거 정확도:** 최근 10개 결정의 정확도는? [Decision Log 확인]
2. **Calibration Check:**
   - 80% 확신 결정들의 실제 성공률: [X%]
   - 90% 확신 결정들의 실제 성공률: [Y%]
3. **겸손 질문:** 내가 틀릴 가능성은? [X%]

### 2.6 집단 사고 (Herding)
**질문:** 다른 투자자들이 모두 그렇게 하니까 따라가는가?

- [ ] ✅ 아니오 - 독립적 판단
- [ ] ⚠️ 예 - 다음을 수행하시오:

**만약 예라면:**
1. **Consensus Check:** 현재 시장 컨센서스는? [Bull/Bear, Buy/Sell 비율]
2. **역발상 질문:** 만약 모두가 틀렸다면? 그 시나리오에서 내 손실은?
3. **독립 판단:** 아무도 이 종목을 사지 않아도 살 것인가? [Yes/No]

## PART 4: 논리 검증 (Logic Validation)

### 4.1 투자 논리 명확성
**핵심 논리 (한 문장으로):**
```
[이 종목을 매수/매도하는 이유를 한 문장으로]
```

**논리 구조:**
1. **병목/우위:** [구조적 병목 또는 경쟁 우위는?]
2. **수익 전환:** [병목이 어떻게 기업 이익으로 전환되는가?]
3. **지속성:** [이 우위는 얼마나 지속되는가?]

### 4.2 논리 상태 판정 재확인
**현재 판정:** [LOGIC STRONGER / INTACT / WEAKENING / BROKEN]

**증거 품질 체크 (STRONGER 판정 시 필수):**
- [ ] 회사 공시/IR 직접 언급
- [ ] 구속력 있는 계약 (MOU 아님)
- [ ] Tier 1 출처 2개 이상
- [ ] 재무제표 개선 확인

**수익 전환 구조 체크:**
- [ ] 가격 결정력 확인
- [ ] 비용 회수 구조 명확
- [ ] CAPEX 부담 < 매출 증가

**지속성 체크:**
- [ ] 2~3분기 이상 지속 가능
- [ ] 경쟁사 진입장벽 존재
- [ ] 고객 다각화 (단일 의존도 <30%)

**판정:** 위 체크리스트 통과? [Yes/No]

### 4.3 대안 가설 (Alternative Hypotheses)
**질문:** 내 논리가 틀렸을 가능성은?

**Alternative Hypothesis 1:**
```
[내 논리와 정반대 시나리오]
발생 확률: [%]
발생 시 영향: [-X%]
```

**Alternative Hypothesis 2:**
```
[다른 해석 가능성]
발생 확률: [%]
발생 시 영향: [-X%]
```

**반증 조건:** 다음 중 하나 발생 시 논리 재검토
- [ ] [조건 1]
- [ ] [조건 2]
- [ ] [조건 3]

## PART 5: 밸류에이션 검증 (Valuation Validation)

### 5.1 Expected Return 재계산
**12개월 시나리오 분석:**

```

- 가정: [시나리오]
- 목표가: $[값] ([+/- Y%])

Base Case (확률: [X]%):
- 가정: [시나리오]
- 목표가: $[값] ([+/- Y%])

Bull Case (확률: [X]%):
- 가정: [시나리오]
- 목표가: $[값] ([+/- Y%])

- [ ] Expected Return >30%: 강력 매수

**현재 Expected Return:** [+/- X%]
**판정:** [위 기준에 부합하는가?]

**Fair Value 추정:** $[값]

**실패 시나리오 1:**
```

예방 방법: [구체적 행동]
```

**실패 시나리오 2:**
```
[두 번째 실패 원인]
발생 확률: [X%]
예방 방법: [구체적 행동]
```

**실패 시나리오 3:**
```
[블랙스완 리스크]
발생 확률: [X%]
대응 방법: [구체적 행동]
```

### 6.2 Exit Plan (탈출 계획)
**질문:** 내가 틀렸다는 것을 어떻게 알 수 있는가?

**Exit Trigger 1 (논리 붕괴):**
```
조건: [구체적 관찰 가능한 신호]
발생 시 행동: [즉시 매도 / 검토 / 축소]
```

**Exit Trigger 2 (가격 손절):**
```
조건: [현재가 대비 -X% 하락]
발생 시 행동: [손절 / 재평가]
최대 허용 손실: $[금액] 또는 [-X%]
```

**Exit Trigger 3 (시간 제한):**
```
조건: [X개월 내 논리 진전 없음]
발생 시 행동: [재평가 / 청산]
```

### 6.3 수면 테스트
**질문:** 이 포지션 크기로 편안하게 잠을 잘 수 있는가?

- [ ] ✅ 예 - 적절한 크기
- [ ] ⚠️ 아니오 - **포지션 축소 필요**

**만약 아니오라면:**
```
불편한 이유: [기록]
편안한 크기: [X%] (현재 목표: [Y%])
조정: 목표 비중을 [X%]로 하향
```

## PART 7: 포트폴리오 수준 영향 (Portfolio Impact)

### 7.1 집중도 리스크

- 최대 단일 종목 비중: [X%] ([TICKER])

- 단일 테마 비중: [Z%] ([테마])

**판정:**
- [ ] ✅ 모두 한도 내 (종목 <20%, 섹터 <35%, 테마 <60%)
- [ ] ⚠️ 한도 초과 - **조정 필요**

### 7.2 상관관계 리스크
**기존 보유 종목과의 상관관계:**

| 기존 종목 | 60일 상관계수 | 리스크 |
|---------|------------|-------|
| [TICKER1] | [0.XX] | [낮음/중간/높음] |
| [TICKER2] | [0.XX] | [낮음/중간/높음] |

**판정:**
- [ ] ✅ 평균 상관관계 <0.7
- [ ] ⚠️ 평균 상관관계 >0.7 - 집중도 위험

### 7.3 유동성 리스크
**해당 종목 유동성:**
- 평균 일일 거래량: [X] shares
- 평균 일일 거래 금액: $[X]
- 내 포지션 크기: $[Y]

**판정:**
- [ ] ✅ 비율 <5% - 유동성 충분
- [ ] ⚠️ 비율 5-10% - 주의
- [ ] ❌ 비율 >10% - 유동성 위험

## PART 9: 최종 판정 (Final Decision)

### 9.1 체크리스트 종합 점수

**필수 항목 (하나라도 실패 시 결정 보류):**
- [ ] 인지 편향 체크 통과
- [ ] Tier 1 출처 2개 이상
- [ ] 논리 상태 판정 타당
- [ ] Expected Return 기준 충족
- [ ] Exit Plan 수립 완료
- [ ] 포트폴리오 한도 내
- [ ] 수면 테스트 통과

**판정:**
- [ ] 7/7 통과 → **실행 승인**
- [ ] 6/7 통과 → **조건부 승인** (미통과 항목 해결 후)
- [ ] ≤5/7 통과 → **결정 보류** (재검토 필요)

**Decision:** [실행 / 보류 / 수정]
