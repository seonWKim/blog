## 7. 포지션 사이징 수학 (Kelly Criterion)

### Kelly Criterion 적용 예시

#### GEV 계산:

```
시나리오 확률:
- Bull: 25% 확률, +38% 수익
- Base: 50% 확률, -17% 수익
- Bear: 25% 확률, -54% 손실

기대값 = 0.25×0.38 + 0.5×(-0.17) + 0.25×(-0.54) = -0.04 (-4%)

Win Rate = 25% (Bull만)
Avg Win = 38%
Loss Rate = 75%
Avg Loss = (0.5×17 + 0.25×54) / 0.75 = 29.3%

Kelly = (0.25 × 0.38 - 0.75 × 0.293) / 0.38 = -0.33

결론: Kelly Criterion이 음수 → 이론적으로 매수 금지
현실 조정: 논리 STRONGER + 장기 구조적 우위 고려 → 최소 비중 유지 (10%)
```

**중요:** Kelly Criterion이 음수일 때는 이론적으로 포지션을 가져서는 안 됨. 그러나 투자 체크리스트는 논리 우선 원칙을 따르므로, LOGIC STRONGER 종목은 최소 비중 유지. 단, 비중 증액은 제한.

#### TER 계산:

```
시나리오 확률:
- Bull: 20% 확률, +17% 수익
- Base: 50% 확률, -22% 수익
- Bear: 30% 확률, -54% 손실

기대값 = 0.20×0.17 + 0.5×(-0.22) + 0.30×(-0.54) = -0.24 (-24%)

Kelly = 음수

결론: Kelly Criterion이 음수 → 이론적으로 매수 금지
현실 조정: 논리 STRONGER + HBM 병목 지속 → 최소 비중 유지 (15%)
```
