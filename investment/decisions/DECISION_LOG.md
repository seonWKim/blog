**범례:**
- ER = Expected Return (예상 수익률)
- Pre = Pre-Decision Score (사전 체크리스트 점수)
- ✅ = 예상보다 좋음 (>120%)
- ✔️ = 예상대로 (80-120%)
- ❌ = 예상보다 나쁨 (<80%)
- ⏳ = 아직 평가 안 됨
- 🔄 = 진행 중
